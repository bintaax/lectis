##Lectis

> Projet final dans le cadre de la formation au Titre Professionnel Développeur Web et Web Mobile

Lectis est une librairie en ligne (e-commerce) permettant aux utilisateurs de pouvoir consulter les livres, gérer leur panier et leurs commandes depuis un espace personnnel.

-----

Technologies utilisées : 

Front-end : HTML5, CSS3, TailwindCSS, Flowbite
Back-end : PHP +8, Symfony
Base de données : MySQL
